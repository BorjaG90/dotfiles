# ZSH

## Dependencias

- zsh instalado
- starship instalado

## Estructura

aliases: Fichero ubicado en ~/.config/zsh o $HOME/.config/zsh
    Contiene alias de funciones habituales
exports: Fichero ubicado en ~/.config/zsh o $HOME/.config/zsh
    Contiene exports habituales (paths, etc.)
.zshrc: Fichero ubicado en ~ o $HOME
